# KFACheckin

## 摘要

為在社團中，每次有課程或活動需要點名時，每個人都需要動手簽到及簽退，會用到一定的時間。

這邊運用Android手機NFC的讀卡功能，讀取每個人學生證或悠遊卡的卡號，只需刷過去，即可完成簽到及簽退。
